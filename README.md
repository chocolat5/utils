# Utils

Collection of reusable code utilities by [@chocolat5](https://github.com/chocolat5).

- Copy-paste ready
- Fully typed with TypeScript
- Customizable and framework-agnostic
- Well-documented with examples

> These utils are designed to be copied into your projects and customized to fit your needs.

### [hono-og-image](./hono-og-image)

Dynamic OG image generator for Cloudflare Workers + Hono

## Usage

Each directory contains detailed documentation and usage examples. Simply copy the files you need into your project and customize them according to your requirements.
